Project 4: Math Inheritance

Involves designing one parent class, and one child class of that parent class using inheritance. The parent class should be named IntMath,and the child class should be named FloatMath. The child should inherit from the parent, and both classes should have a contructor and 2 methods: add() and mult().

Parent Class
  • Named IntMath
  • Has a constructor that initializes the variable num to parameter passed into it
  • Has method add() that returns sum of num and one int parameter.
  • Has mult() method that returns product of num and one int parameter.
Child Class
  • Named FloatMath
  • Inherits from the parent class. Has a constructor that calls the constructor of the parent
  • Has method add() that overrides parent add() method. Should return parent add() call and sum of num and the parameter converted to a float side by side in a string. See sample output for clarifcation.
  • Has mult() method that that overrides parent add() method. Should return parent product() as well as float casted product of num and the parameter side by side in a string.

Sample Output


parent = IntMath(5)
child = FloatMath(5)
print(parent.add(4))
#Output: 9
print(child.add(4))
#output: 9 9.0

The parent's add() returns num (5) + 4, which is 9. The child's add() returns the parent call alongside the decimal addition of 5 + 4 = 9.0.

Sample Output


parent = IntMath(5)
child = FloatMath(5)
print(parent.mult(4))
#Output: 20
print(child.mult(4))
#output: 20 20.0

The parent's mult() returns num (5) * 4 = 20. The child mult() returns the parent call alongside 5 * 4 = 20 converted to a float.

Note: When looking at the test cases, assume p = IntMath(), the parent object, and c = floatMath(), the child object.
