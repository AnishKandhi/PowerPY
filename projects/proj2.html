Project 2: Star Pattern

This project involves creating a program that writes a function called createStars(), which produces a pattern of astricks (*) based on the function parameter. The function takes one integer parameter, n. If the parameter is even, the function outputs rows of stars with the first row having one star, the last having n stars, and the amount of stars increments for each subsequent row. This will form a right triangle.
If n is odd, the first row should have n stars, the last should have one, and each subsequent row should decrement the number of stars it has. This will form an inverted right triangle. (Hint: the empty print() function with no parameters creates a new line).

createStars(4)

In this case, n = 4, which is an even number. So the stars will be printed from 1 . . . n, incrementing each row.

createStars(5)

In this case, n = 5, which is odd. So the stars will print backwards, from n . . . 1, decrementing each row.
