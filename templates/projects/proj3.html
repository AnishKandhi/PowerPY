Project 3: Quantity Tracker

This project involves writing code for a function, trackNums(nums), which takes in one list of integers named nums. Your task is to iterate through the list, and store the amount of times each number appears in a new dictionary. In the dict, the key for each pair should be the number, and the value should be the frequency of that number. The function should return the new, final dictionary.

trackNums([1, 1, 4, 5])

In the list, '1' appears twice, while '4' and '5' appear once. So the key-value pairs are 1:2, 4:1, and 5:1.

trackNums([0, 3, 4, 3, 3, 0, 10])

'0' appears twice, '3' appears 3 times, and '4' and '10' appear once. The pairs are 0:2, 3:3, 4:1, 10:1.
