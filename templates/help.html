What is this site?

This application was made to provide a simple, concise introduction to the python programming language for beginners looking to enter fields like web development, AI, or data science.

What learning options are there?

Users may click on the tiles to learn different programming concepts. Each of these tiles provides a lesson with concise explanations and code snippets to provide python examples. At the end of each lesson, there is a multiple choice quiz, which provides a score and an option to review missed questions. After the end of section, users can also complete the project, which provides instructions on solving a coding problem and an built-in coding editor for users to write their solutions in. Users can run their code in the browser, and feedback is given with which test cases the user's code passed and failed.

Quiz Scoring Delay Note

After a period of inactivity, the server that scores the quizzes and authenticates users may sleep temporarily. Therefore, be patient with your first quiz submission or login attempt, as it can sometimes take 10-30 seconds before the server activates and the delays go away.
