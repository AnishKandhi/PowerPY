Project 1: Average Computing

This project involves creating a program that computes the average, or arithmetic mean, of user entered values. You will need to prompt the user to enter three different values, and print the mean of these values to the console. This will involve type conversions, basic math, and user input handling.
(Hint: Try and except blocks will be needed to make sure the values the user is entering is able to be casted to an integer. In other words, make sure they are ints or floats, not strings). -1 should be printed in the case that the user enters a value that can not be converted to an int and divided with, like a character or word.

Sample Output

The user is prompted to enter three numbers. The user entered 2, 6, and 7. The average of them is 5, which is printed to the console.

Sample Output

The user is prompted to enter three numbers. The user entered 10, 7, 3. The average of them is 6.667, which is printed to the console.

Note: -1 should be returned if the user enters a non-number, like a string or int. The test cases will test to see if a ValueError is caught.
